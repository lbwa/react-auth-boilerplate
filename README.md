> This project is used to build a front-end authentication boilerplate.

## Client

- install all dependencies

  ```bash
  yarn bootstrap --registry https://registry.npmjs.org/
  ```

- start client at the `http://localhost:3000`

  ```bash
  yarn start:client
  ```
